# meritocracia API

## Instalacion 

Clonar el repositorio

`https://github.com/SocialMeetPeru/meritocracia-api.git`

`cd meritocracia-api/`

Crear una base de datos con el nombre 

`meritocracia_api`

Importar el archivo meritocracia_api.sql a la BD creada

`/src/auth/bd/meritocracia_api.sql`

Poner los datos de acceso de la BD en el archivo 

`/src/auth/config/conn.php`

Ahora si corremos el server en el puerto 8083
 
`php -S localhost:8083`
